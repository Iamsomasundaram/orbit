# ORBIT Milestone 0

Milestone 0 defines the product, architecture, contracts, evaluation strategy, and repo blueprint for ORBIT. This directory is the approval gate for all deeper implementation work.

Document map:

- `brd.md` -> business requirements document
- `prd.md` -> product requirements document
- `system-architecture.md` -> platform modules, services, runtime boundaries
- `agent-architecture.md` -> specialist reviewer definitions and orchestration contracts
- `conflict-detection.md` -> structured conflict strategy and debate triggers
- `scoring-model.md` -> scoring dimensions, weighting, and recommendation tiers
- `evaluation-rubric.md` -> evaluation rubric, stability criteria, and quality bars
- `golden-portfolio-format.md` -> canonical portfolio format and starter dataset contract
- `repo-structure.md` -> repository layout and ownership model
- `milestone-plan.md` -> phased delivery plan with parallel workstreams
- `review-pack.md` -> Ralph Framework milestone review pack

Approval rule:

- Milestone 0 must be reviewed and accepted before Milestone 0.5 implementation expands beyond the current scaffold.
